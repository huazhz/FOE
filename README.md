# FOE
based on ThinkPHP 3.2.3 
### Free Online Education 
FOE是一款在线教育系统 在传统MOOC的功能基础上增加可直播、Online Judge 以及实时聊天

## 环境要求
php >=5.6 mysql>=5 apahce>=2.4

## 功能介绍

- 拥有三个管理板块,分别为学员中心 教师中心 管理员后台
- 论坛功能
- 站内邮箱功能
- 实时聊天功能
- 直播功能
- Online Judge功能
- 手册、新闻、消息日志等功能
- 权限功能
